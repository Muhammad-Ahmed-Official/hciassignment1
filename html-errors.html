Sample questions: fix common HTML structure and accessibility mistakes
UI Element / Issue Wrong HTML (with mistake) Correct UI Element Correct HTML
Navigation menu has a link directly inside ul (missing li) 
<nav>
  <ul>
    <a href="index.html">Home</a>
  </ul>
</nav>
 Navigation list where each link is inside a list item 
<nav aria-label="Primary navigation">
  <ul>
    <li><a href="index.html">Home</a></li>
  </ul>
</nav>
Form label not associated with control (for ≠ id) 
<label for="msg">Message</label>
<textarea id="message"></textarea>
 Label correctly connected to input 
<label for="message">Message</label>
<textarea id="message"></textarea>
Heading hierarchy is incorrect (skips a level) 
<h1>Main Title</h1>
<h3>Sub Title</h3>
 Headings follow proper hierarchy 
<h1>Main Title</h1>
<h2>Sub Title</h2>
Table missing caption and th 
<table>
  <tr><td>Model</td><td>Predicts</td></tr>
</table>
 Accessible table with caption + headers 
<table>
  <caption>Models and what they predict</caption>
  <thead>
    <tr>
      <th scope="col">Model</th>
      <th scope="col">Predicts</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">KLM</th>
      <td>Execution time</td>
    </tr>
  </tbody>
</table>
aria-current used incorrectly (multiple current pages) 
<nav>
  <ul>
    <li><a href="index.html" aria-current="page">Home</a></li>
    <li><a href="cv.html" aria-current="page">CV</a></li>
  </ul>
</nav>
 Only active page marked as current 
<nav>
  <ul>
    <li><a href="index.html">Home</a></li>
    <li><a href="cv.html" aria-current="page">CV</a></li>
  </ul>
</nav>
Image missing alt attribute 
<img src="profile.jpg">
 Provide descriptive alt text 
<img src="profile.jpg" alt="Profile photo of Muhammad Ahmed">
Empty link with no text 
<a href="contact.html"></a>
 Link should have visible text or aria-label 
<a href="contact.html">Contact</a>
Form missing fieldset and legend 
<form>
  <label for="name">Name</label>
  <input type="text" id="name">
</form>
 Group related controls with fieldset + legend 
<form>
  <fieldset>
    <legend>Personal Info</legend>
    <label for="name">Name</label>
    <input type="text" id="name">
  </fieldset>
</form>
Missing lang attribute on <html> 
<html>
<head><title>Page</title></head>
<body>Content here</body>
</html>
 Include lang attribute to specify document language 
<html lang="en">
<head><title>Page</title></head>
<body>Content here</body>
</html>
Incorrect semantic element for main content (using div only) 
<div id="main">Content here</div>
 Use semantic <main> element 
<main>Content here</main>
